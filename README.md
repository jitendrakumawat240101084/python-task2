### ✅ Task 1: Check Whether a Number is Even or Odd

#### 📖 Problem Statement

Write a Python program that: - Takes a number as input from the user. -
Checks whether the number is even or odd. - Displays the result.

#### 💻 Code

``` python
# Task 1: Check if the number is even or odd

# taking input from the user
number = int(input("enter the number: "))

# performing operation
if number % 2 == 0:
    print(number, " is an even number ")
else:
    print(number, " is an odd number ")
```

### ✅ Task 2: Sum of Integers from 1 to 50 Using a Loop

#### 📖 Problem Statement

Write a Python program that: - Calculates the sum of numbers from 1 to
50. - Uses a `for` loop. - Prints the final result.

#### 💻 Code

``` python
# Task 2: Sum of Integers from 1 to 50 Using a Loop

# initialize sum variable with 0 value
total = 0

# using for loop range 1 to 50 calculate the sum
for i in range(1, 51):
    total += i

print("The sum of numbers from 1 to 50 is:", total)
```
